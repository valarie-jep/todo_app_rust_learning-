Rust To-Do List Application

This project is a command-line To-Do List application built using Rust.
It was developed as part of a learning exercise to understand a new programming language with the help of AI, while building a practical project.

Project Description

The application allows users to:

Add tasks

View all tasks

Mark tasks as completed

Delete tasks

Save tasks to a file so they persist between runs

Tasks are stored using JSON format, and each task can include a priority level and an optional due date.

Learning Objectives

Through this project, we aimed to learn:

Basic Rust syntax and structure

Structs, enums, and pattern matching

File handling in Rust

Using external crates (serde, serde_json)

Building and running Rust projects with Cargo

Refactoring code into functions

Using Git and GitHub for version control
